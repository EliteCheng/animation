# Animation Learning

## CSS3 Animation

### how to start
#### install dependence
```bash
git clone https://github.com/EliteCheng/animation.git
cd animation
yarn install
```
#### run
modify project path config
```git
# config project path. open ./webpack-config/webpack.base.js 
  const {
      mainPath,
      htmlPath,
- } = require('../svg-animation/config')
+ } = require('../css3-animation/config')
```
run project
```bash
yarn run dev
yarn run prod # build 
```
